HTML codes with best practices of SEO...

The purpose of any website is to drive traffic. And a vast portion of those visitors is coming from search engines. Perhaps one of the best ways to engage an audience is through using HTML tags for SEO. There's always something new to learn in HTML, whether you're a coding newbie or an experienced pro.

The code may be correct but there are still few things that are kept in mind for SEO

One code must contain only one h1 tag

h1 tag should be followed by h2,h3,h4...h6 in a sequential way.

Use Strong tag instead of bold tag

Use Emphesize tag instead of italic tag
